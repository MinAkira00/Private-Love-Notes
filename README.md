## 🏗️ Estructura del Proyecto

```
Amor/
├── backend/           # API Backend (Node.js/Express)
├── web-frontend/      # Aplicación Web (React/Vite)
├── mobile-app/        # Aplicación Móvil (React Native)
├── desktop-app/       # Aplicación Escritorio (Electron)
├── shared/           # Código compartido entre plataformas
└── docs/             # Documentación
```

## 🚀 Tecnologías

### Backend
- Node.js + Express
- MongoDB/SQLite
- JWT Authentication
- RESTful API

### Web Frontend
- React 18
- Vite
- Tailwind CSS
- Local Storage / IndexedDB

### Mobile App
- React Native
- Expo (para desarrollo rápido)
- AsyncStorage

### Desktop App
- Electron
- React (compartido con web)

### Shared
- TypeScript interfaces
- Utilidades comunes
- Validaciones
